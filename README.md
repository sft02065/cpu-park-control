## Windows 悄悄"关掉"了你的 CPU

内核暂停是 Windows 的一项省电特性，简单来说就是：系统觉得你的 CPU 暂时用不了那么多核，就把多余的核"挂起"进入低功耗睡眠状态。听起来挺合理，但问题在于——Windows 对这个机制太激进了，就算是默认的"高性能"电源计划，也会在你稍微空闲一会儿之后把核心停掉。

等到你突然需要 CPU 发力的时候，系统要先把这些睡着的核唤醒，这个过程就产生了延迟。对游戏来说就是帧率抖动，对日常使用来说就是那一瞬间的"反应迟钝"。

Intel 后来也意识到这个问题，从 Skylake 一代开始把内核暂停控制权从操作系统移到了芯片硬件层——算是侧面印证了 Bitsum 当年的判断。
